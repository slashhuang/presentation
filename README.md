# architecture-fe

a boilerplate for fe architecture

# command

### dev
```bash
    npm run dev
    open localhost:9000
```

# main functionalities

- npm scripts [dev product deploy]

- babel [ES6 ES5]

- webpack [图片、css、js等处理]

- jest[单元测试]

- vue/react 框架

# file structure

```bash
|- src
|- |- app
|- |- components
|- |- global
|- |- |- reset.css
```
